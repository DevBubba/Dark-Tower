# <a id="description"></a> 🎮 〢 Dark Tower


### The Princess is at the Top of the Tower. You Need to Get There as Fast as You Can and Try to Save Her! Try to Beat Your Best Time and Get the Fastest Time You Can. The World Record Run is 33.96 Seconds Set by Quinton Hall! Can You Beat It? Find Out by Playing the Game for Yourself!

# <a id="gamecontrols"></a> 🕹️ Game Controls


| Description                                                | Keys                                                                 |
| ---------------------------------------------------------- | -------------------------------------------------------------------- |
| Start Game                                                 | <kbd>S</kbd>                                                         |
| Sprint                                                     | <kbd>X</kbd>+<kbd>Arrow Key (Left Or Right)</kbd>                    |
| Jump                                                       | <kbd>Up</kbd>+<kbd>Arrow Key</kbd>                                   |
| Move Left                                                  | <kbd>Left</kbd>-<kbd>Arrow Key</kbd>                                 |
| Crouch                                                     | <kbd>Down</kbd>-<kbd>Arrow Key</kbd>                                 |
| Move Right                                                 | <kbd>Right</kbd>-<kbd>Arrow Key</kbd>                                |
| Slide                                                      | <kbd>X</kbd>+<kbd>Arrow Key (Left Or Right)</kbd>                    |
| Open Note                                                  | <kbd>Z</kbd>                                                         |



# <a id="keyboardshortcuts"></a> ⌨️ Keyboard Shorcuts


| Description                                                | Keys                                                                 |
| ---------------------------------------------------------- | -------------------------------------------------------------------- |
| Skip To End Of Game Credits Easter Egg                     | <kbd>Shift</kbd>+<kbd>}</kbd>                                        |
| Skip To End Of Game Easter Egg                             | <kbd>Shift</kbd>+<kbd>{</kbd>                                        |
| Skip Intro Of Game Start                                   | <kbd>Space</kbd>                                                     |
| Skip End Of Game Scene                                     | <kbd>Space</kbd>                                                     |
| Restart Game                                               | <kbd>R</kbd>                                                         |
| Skip Scene                                                 | <kbd>Space</kbd>                                                     |
| Main Menu                                                  | <kbd>Q</kbd>                                                         |
| Open Note                                                  | <kbd>Space</kbd>                                                     |
